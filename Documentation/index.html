A) Installation - top

To add a WordPress Plugin using the built-in plugin installer:

  1. Go to Plugins > Add New.
  2. Under Upload tab, click on "Choose File" and choose nicescrollbar-wp.zip from your hard drive.
  3. Click Install Now to install the plugin.
  4. A popup window will ask you to confirm your wish to install the Plugin.
  5. If this is the first time you've installed a WordPress Plugin, you may need to enter the FTP login credential information. If you've installed a Plugin before, it will still have the login information. This information is available through your web server host.
  6. Click Proceed to continue with the installation. The resulting installation screen will list the installation as successful or note any problems during the install.
  7. If successful, click Activate Plugin to activate it, or Return to Plugin Installer for further actions.

To add a WordPress Plugin using the built-in plugin installer:

  1. Login to your hosting via your FTP client.
  2. Navigate to your WordPress plugin directory (usually it's wp-content/plugins)
  3. Extract nicescrollbar-wp.zip and copy nicescrollbar-wp folder to plugin directory on your host.
  4. Login to your WordPress Admin dashboard and go to Plugins page.
  5. You should now see Nicescrollbar WP in the list of plugins. Click Activate Plugin to activate it.

B) How to use - top

After activating the plugin you will find Nicescrollbar WP Options Menu in your dashboard.

Go to Nicescrollbar WP Options. You will find options to customize the scrollbar.

Available options are

  1. Scrollbar Color – Change scrollbar color in hex, default is "#000000".
  2. Minimum Scrollbar Opacity – Change opacity when scrollbar is inactive (scrollabar "hidden" state), range from 1 to 0, default is 0 (hidden).
  3. Maximum Scrollbar Opacity – Change opacity when scrollbar is active (scrollabar "visible" state), range from 1 to 0, default is 1 (full opacity).
  4. Scrollbar Hiding Delay - Set the delay in microseconds to fading out scrollbar default: 400.
  5. Scrollbar Width – Scrollbar width in pixel, default is "5px".
  6. Scrollbar Border Color – Color of the border. default is "#fff".
  7. Scrollbar Border Width – Border Width in pixel. Default value is "0px".
  8. Scrollbar Border Radious – Border radius in pixel for scrollbar, default is "0px".
  9. Scrolling Speed – Scrolling speed, default value is 60.
  10. Scrolling speed with mouse wheel – Scrolling speed with mouse wheel, default value is 40 (pixel).
  11. Enable cursor-drag - Enable cursor-drag scrolling like touch devices in desktop computer (default:false).
  12. Use Hardware Acceleration - use hardware accelerated scroll when supported (default:true).
  13. Enable Boxzoom - enable zoom for box content (default:false).
  14. Display "grab" icon - Display "grab" icon for div (This will available only when Enable cursor-drag is enabled).
  15. Auto Hide Scrollbar - Enable/Didable autohide of the scrollbar.
  16. Enable Bouncescroll – Enable scroll bouncing at the end of content as mobile-like (only hw accell) (default:false).
  17. Enable Horizontal Scrollbar - Enable Horizontal Scrollbar (default:true).
  18. Vertical Scrollbar Alignment - Alignment of vertical scrollbar (defaul:"right").
  19. Horizontal Scrollbar Alignment - Alignment of horizontal scrollbar (defaul:"bottom").
  20. Enable Scroll With Mouse Wheel - Enable/Disable Scroll With Mouse Wheel (default:true).
  21. Enable Smoothscroll – Scroll with ease movement (default:true).

Just change the options as you like. And enjoy!
